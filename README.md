# Panda-Dataframe-Exercises
A collection of examples where I applied pandas dataframes to various files for data analyses.

By following an online course on Udemy, I learned basic and moderate principles of accessing CSV files and doing data analytics. In two scenarios, I accessed customer information to perform analytics such as highest purchases, specific job titles, or credit card information.

The Salaries Pandas Exercise focuses on assessing a CSV file of employees with their salaries, base pay, job title, etc. The CSV file for this exercise is called Salaries.csv.

The E-commerce Purchases Exercise consists of accessing the purchase information of customers. The CSV file, called Ecommerce_Purchases.csv, contains additional information such as credit card information, address information, and more.
